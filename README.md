# WhiteList4qq 1.20+
让你的群友申请白名单 支持1.20.1+
根据另一个同名插件修改而来：
https://github.com/DreamVoid/Whitelist4QQ
跟另一个同名插件功能一样，受其启发，自己写了个1.20.1的版本给自己服务器用，并使用mysql存储绑定的数据

需要MiraiMC作为前置，但如你所见，没有用到miraimc的绑定数据库
